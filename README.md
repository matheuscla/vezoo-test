This is a simple online code Editor developed in React.


## How to run
- Clone the project, run `yarn` to install the dependencies and `yarn start` to start the project. The project will run on http://localhost:3000


## Testing
- Run `yarn test`


## Technologies
- React
- Redux
- Styled-components
- Axios
- Prismjs
- Jest
